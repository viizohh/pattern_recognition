# SUSPICIOUS PATTERNS

## 1. Command-and-Control (C2) Communication

**Pattern:** Beaconing to api-check.malware-c2.com
- **Frequency:** Every 60 seconds (15 connections over 15 minutes)
- **Consistency:** Extremely regular intervals (coefficient of variation < 0.1)
- **Domain Reputation:** Domain name suggests checking for updates/commands
- **Risk Assessment:** CRITICAL

**Analysis:** This is classic C2 beaconing behavior. The infected device is:
1. Checking in with attacker infrastructure regularly
2. Likely receiving commands or exfiltrating small amounts of data
3. Using a domain that appears to be purpose-built for malware operations

**Immediate Action Required:** Isolate device from network immediately.

## 2. Domain Generation Algorithm (DGA)

**Pattern:** High-entropy domain x8j3k2m9q1z7p4.net
- **Shannon Entropy:** 4.3 (threshold: 4.0)
- **Characteristics:** Random character distribution, no pronounceable segments
- **Queries:** 8 queries, all failed to resolve
- **Risk Assessment:** HIGH

**Analysis:** The domain exhibits characteristics consistent with algorithmically generated domains used by malware to:
1. Evade domain blacklists
2. Establish fallback C2 channels
3. Make takedown efforts more difficult

The failed DNS resolutions suggest the malware is cycling through generated domains looking for active C2 infrastructure.

## 3. Network Reconnaissance

**Pattern:** Port scanning from 192.168.1.100 → 10.0.0.50
- **Ports Scanned:** 20-40 (21 consecutive ports)
- **Duration:** 5.2 seconds
- **Success Rate:** 0% (all connection attempts failed)
- **Target:** Internal host 10.0.0.50
- **Risk Assessment:** HIGH

**Analysis:** This is clear reconnaissance behavior indicating:
1. The malware is attempting lateral movement
2. Target (10.0.0.50) is being probed for vulnerable services
3. Systematic scanning of common service ports:
   - Port 21 (FTP)
   - Port 22 (SSH)
   - Port 23 (Telnet)
   - Port 25 (SMTP)
   - Ports 26-40 (various services)

The failed connections suggest the target may be hardened or offline, but the attempt demonstrates hostile intent.

## 4. Excessive Third-Party Tracking

**Pattern:** 25 third-party domains loaded from news-site.com
- **Primary Site:** news-site.com
- **Third-Party Count:** 25 distinct domains
- **Risk Assessment:** MEDIUM (not directly related to malware, but privacy concern)

**Analysis:** While not directly related to the malware infection, this indicates:
1. Significant browser fingerprinting exposure
2. Potential for malicious ad injection
3. Increased attack surface through third-party scripts

Many compromised websites inject malicious tracking to monitor user behavior.
